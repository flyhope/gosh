# gosh
terminal ui ssh client

## Be dependent on program
1. ssh
2. sshpass (if use auto input password)
3. zssh (if use rz/sz)

## Usage
1. copy `config.simple.xml` to `config.xml`
2. run gosh

## Support
* [x] custom command path
* [x] custom username
* [x] custom port
